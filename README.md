## Active Projects

**[Concept Coherence Metric (CCM)]()**  
A framework for evaluating whether sparse autoencoder features in LLMs correspond to coherent, human-interpretable concepts. Built at SRI International / NuSCI Group.

**[LLM Web Agent Security SoK]()**  
A systematization of 50+ attack techniques against autonomous web-browsing agents — organized along a perception → reasoning → action pipeline. With Dr. Taha Khan @ CMU CyLab.

**[Post-Quantum Cryptography over BLE/NFC]()**  
Empirical evaluation of Kyber and SABER variants on constrained BLE and NFC channels. Targeting ACM WiSec.
