**[Knowledge Retention in Fine-Tuned LLMs](https://aclanthology.org/2025.findings-emnlp.589/)**  
`LUMS TPI Lab · NYUAD · EMNLP 2025`

Comprehension-based tasks (QA) achieve **48% knowledge retention** vs **17%** for surface-level mapping tasks (translation). Scaling improves retention from 38% → 72% systematically across 5 model families (1.5B–72B).
